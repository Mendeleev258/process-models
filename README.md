**Задача 4:**

В информационной системе присутствует имя, пол, дата рождения и домашний адрес каждого пациента. Всякий раз, когда врач осматривает больного, явившегося к нему на прием, или сам приходит к нему на дом, он записывает дату и место, где проводится осмотр, симптомы, диагноз и предписания больному, проставляет имя пациента, а также свое имя. Если врач прописывает больному какое-либо лекарство, заносится название лекарства, способ его приема, словесное описание предполагаемого действия возможных побочных эффектов. 

Спроектировать систему, которая позволяет:
1. по заданной дате определить количество вызовов в этот день
2. позволяет определить количество заболевших данной болезнью
3. по заданному лекарству определить его побочный эффект
4. предоставить возможность добавления нового лекарства с описанием его свойств
